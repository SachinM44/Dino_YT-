## Getting Started

```bash
git clone https://github.com/SachinM44/Dino_YT-.git
cd Dino_YT-
npm install
npm run dev
```

Opens at http://localhost:5173
run npm run dev -- --host to run on network
and check in mobile by using that ip address : 
ex:  http://192.168.100.197:5173/ (copy the url and past it on target mobile device browser)
